Introducción

Python es un lenguaje de programación interpretado, de alto nivel y propósito general.

Fue creado por Guido van Rossum y lanzado en 1991.

Python enfatiza la legibilidad del código con su notación clara y concisa.

Se usa en desarrollo web, ciencia de datos, automatización, inteligencia artificial y más.

Es una excelente opción para principiantes y desarrolladores avanzados.

print("Hola, mundo")

Python tiene una gran comunidad de usuarios y una amplia gama de librerías.

Permite escribir menos líneas de código para lograr más.

# Esto es un comentario en Python
  • Fácil de aprender
  • Extensibilidad
  • Multiplataforma
Sintaxis básica

Python no usa llaves sino indentación para definir bloques de código.

Las variables se declaran automáticamente al asignar un valor.

x = 10 nombre = "Juan"

Los comentarios se escriben con # al inicio de la línea.

# Esto es un comentario
  • No se necesitan punto y coma (;)
  • Indentación importante
  • Case-sensitive
Tipos de datos

Python tiene varios tipos de datos incorporados como:

  • int
  • float
  • str
  • bool
  • list, tuple, dict, set

Ejemplos:

edad = 25 pi = 3.14 es_mayor = True

Las listas son mutables, las tuplas no.

Estructuras de control

Python incluye estructuras como if, for y while.

Ejemplo de if:

if edad >= 18:\n print("Es mayor de edad")

Ejemplo de for:

for i in range(5):\n print(i)

Ejemplo de while:

while x < 10:\n x += 1
  • Condicionales
  • Bucles
  • Break y continue
Funciones

Las funciones se definen con la palabra clave `def`.

Pueden devolver valores con `return`.

def saludar():\n print("Hola") def sumar(a, b):\n return a + b

Se pueden pasar parámetros con valores por defecto.

Ejemplo de llamada:

resultado = sumar(3, 5)
